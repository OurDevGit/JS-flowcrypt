Explaining FlowCrypt webmail permissions

Manage drafts and send emails helps you compose and send encrypted emails.
Read messages allows you to reply to a conversation and open attachments. It also makes setup and use of this plugin a smoother experience.
Nobody, FlowCrypt developers included, is able to access these permissions, they are stored privately in your browser. Learn more about email provider access and security.
